# automation-toolkit-desktop-app
Multi-tool desktop automation software built with Python Tkinter including file organizer, bulk renamer, Excel cleaner, web scraper and weather API tool.

# Automation Toolkit Desktop Application

A multi-tool desktop software built using Python Tkinter that integrates multiple automation utilities into one application.

---

## Features

* Folder Organizer
* Bulk File Renamer
* Excel Data Cleaner
* Web Link Scraper
* Weather API Tool

---

## Technologies Used

* Python
* Tkinter
* Requests
* Pandas
* BeautifulSoup
* OS module

---

## Project Purpose

This project demonstrates real-world Python automation skills including:

* GUI development
* File handling
* API integration
* Data processing
* Web scraping
* Modular software architecture

---

## How It Works

The main GUI acts as a control panel that launches individual tools in separate windows. Each tool performs a specific automation task and provides output to the user.
